# bonsai 画像配置メモ

`/collapse` (EXP-002 / COLLAPSE BONSAI) で使用するステージ画像。
`<img>` で `/hassou-lab/bonsai/...`（basePath 込み）として参照する。
※ `output: 'export'` のため `next/image` は使用しない。

| ファイル名 | ステージ | 状態 |
| --- | --- | --- |
| not_collapsed.png | Stage 0 | OBSERVING（完全体） |
| Early_Collapse.png | Stage 1 | UNSTABLE（初期崩壊） |
| Mid_Collapse.png | Stage 2 | COLLAPSING（構造的崩壊） |
| Late_Collapse.png | Stage 3 | VOID（消滅） |

Stage 4（REBIRTH）は画像なし・種のみ表示。
